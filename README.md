
# PlayListify

## Description

PlayListify is a web app that allows users to convert their Spotify playlists into YouTube playlists. The app uses OAuth2 authentication with Spotify and Google accounts to access user playlists and create corresponding playlists on YouTube.

## Features

- Login with Spotify and Google
- Select songs from your Spotify playlist
- Automatically create a YouTube playlist for the selected songs
- View progress and receive success/error notifications

## Setup

1. Clone the repository
2. Create environment variables for Spotify and Google OAuth
3. Run `npm install` to install dependencies
4. Run `npm run dev` to start the development server
5. Deploy on Vercel or similar platform
